The $$\TeX$$ programming language can do lots of cool things. It natively
supports typesetting for equations, so you can type something like this:

{% highlight tex %}
d = v_0 t + \dfrac{1}{2} a t^2
{% endhighlight %}

#### Output

$$
d = v_0 t + \dfrac{1}{2} a t^2
$$
